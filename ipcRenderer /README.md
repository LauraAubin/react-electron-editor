# React Electron Editor

## Setup

1. Install dependencies:

```bash
$ yarn install
```

2. Run locally:

```bash
$ yarn start
```

3. Build an Electron application:

```bash
$ yarn build
```

4. Find the app in in the `./dist` folder.
